# **App Name**: TitleNote

## Core Features:

- Title List: Home screen displaying a vertical list of user-defined titles.
- Real-time Search: Search titles by name or keyword in real-time, at the top of the home screen.
- Add Title: Add new titles using a '+' button on the home screen.
- Note Creation: Individual title screens where users can add text notes.
- Save Notes: Each text note can be saved to a vertical list within its title.
- Copy to Clipboard: A 'Copy' button allows users to copy the content of a note to the clipboard.
- Dark/Light Mode Toggle: A button allowing users to switch between dark and light mode.

## Style Guidelines:

- Primary color: Soft blue (#A7D1FF), evokes calmness and organization, reflecting the app's note-taking purpose.
- Background color: Light gray (#F0F4F7), to support comfortable reading.
- Accent color: Light orange (#FFC288), highlighting CTAs and interactive elements, creating a balance.
- Body and headline font: 'PT Sans', a versatile sans-serif font, ideal for readability on Android devices, is suitable for both titles and note content.
- Use minimalist icons for add, copy, and delete functions.
- Vertical lists for titles and notes, providing a straightforward navigation.
- Subtle, fast animations for screen transitions.
